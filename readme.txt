How to Run:

You can find the search-engine-ui project repo here: https://github.com/AhmedMahmoud98/search-engine-ui
-------------- Run search-engine-user-interface -----------------

1) install Nodejs
- sudo apt install python-software-properties
- curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
- sudo apt install nodejs

2) install Angular CLI
- npm install -g @angular/cli

3) Build project 
- Open project folder "search-engine-ui"
- Run the following command to build the project:
  ng serve

------------------ Run search-engine-API -------------------
1) install Maven using Eclipse
- Open Eclipse IDE
- Click Help -> Install New Software...
- Click Add button at top right corner
- fill up Name as "M2Eclipse" && Location as "http://download.eclipse.org/technology/m2e/releases"
- click OK

2) Open the project using Eclipse
- Open Eclipse IDE
- Click file -> Import
- Choose Maven -> Existing Maven Project
- Choose The Project Folder "Search-Engine-API" (Eclipse will Download some dependencies at first)

3) Run The project from Class "SearchEngineApiApplication"


------------------ Run search-engine (Crawler-Indexer-PageRank Algorithm) -------------------
1) Open the project using Eclipse
- Open Eclipse IDE
- Click file -> Import
- Choose General -> Existing Project into workspace
- Choose The Project Folder "Search-Engine"

2) Run The project from Class "Main"
